Math & Statistical Functions

Numbers ke saath kaam — data analytics ka core
SUM / SUMIF / SUMIFS
Jodna — condition ke saath ya bina
EXCEL
-- Simple sum
=SUM(B2:B100)

-- Sirf "Delhi" wali rows ka sum
=SUMIF(A2:A100, "Delhi", B2:B100)

-- Delhi AND amount > 5000 wali rows
=SUMIFS(B2:B100, A2:A100, "Delhi", B2:B100, ">"&5000)

-- Is mahine ki sales
=SUMIFS(D2:D100, C2:C100, ">="&DATE(2024,1,1), C2:C100, "<="&DATE(2024,1,31))
✅ Kab Use KaroTotal revenue, category-wise sales, date range mein jodna. SUMIFS sabse zyada use hoti hai — multiple conditions ke saath.
COUNT / COUNTA / COUNTIF / COUNTIFS
Ginana — numbers, text, ya condition ke saath
EXCEL
-- Sirf numbers count
=COUNT(A2:A100)

-- Sab kuch count (text bhi)
=COUNTA(A2:A100)

-- Kitne "Delhi" hain?
=COUNTIF(A2:A100, "Delhi")

-- Delhi mein 5000 se zyada orders
=COUNTIFS(A2:A100, "Delhi", B2:B100, ">"&5000)

-- Blank cells count
=COUNTBLANK(A2:A100)
✅ Kab Use KaroTotal orders, unique entries check, missing data count, condition-wise count.
AVERAGE / AVERAGEIF / MIN / MAX / MEDIAN
Statistical analysis ke liye
EXCEL
-- Average
=AVERAGE(B2:B100)

-- Sirf Delhi ka average
=AVERAGEIF(A2:A100, "Delhi", B2:B100)

-- Sabse badi aur chhoti value
=MAX(B2:B100)
=MIN(B2:B100)

-- Middle value (outliers se better)
=MEDIAN(B2:B100)

-- 2nd highest value
=LARGE(B2:B100, 2)

-- 2nd lowest value
=SMALL(B2:B100, 2)
✅ Kab Use KaroAverage order value, highest/lowest sale, outliers check karna. MEDIAN better hai jab data mein bahut bade ya chhote values ho.
ROUND / INT / MOD
Numbers ko format karna
EXCEL
-- 2 decimal places tak round
=ROUND(A1, 2)

-- Nearest integer tak floor
=INT(A1)

-- Remainder nikalna
=MOD(A1, 2)  -- 0 = even, 1 = odd

-- Percentage round karna
=ROUND(A1/B1*100, 1) & "%"
✅ Kab Use KaroPrice calculations, percentage display, even/odd check. Financial reports mein ROUND zaroori hai.

Text Functions

Messy text data ko clean karna — real job mein roz kaam aata hai
TRIM / UPPER / LOWER / PROPER
Text clean karna — case aur spaces
EXCEL
-- Extra spaces hatao (beginning, end, middle)
=TRIM(A1)

-- UPPERCASE banana
=UPPER(A1)

-- lowercase banana
=LOWER(A1)

-- First Letter Capital (Proper Case)
=PROPER(A1)

-- Clean + Trim ek saath (hidden characters bhi hatata hai)
=TRIM(CLEAN(A1))
✅ Kab Use KaroImport kiye data mein hamesha TRIM lagao. "delhi" aur "Delhi" alag count hote hain — PROPER se fix karo.
LEFT / RIGHT / MID / LEN
Text ke parts nikalna
EXCEL
-- Pehle 3 characters
=LEFT(A1, 3)

-- Aakhri 4 characters
=RIGHT(A1, 4)

-- Beech se characters (position 3 se, 5 characters)
=MID(A1, 3, 5)

-- Text ki length
=LEN(A1)

-- Phone number se last 10 digits
=RIGHT(A1, 10)

-- Email se domain nikalna
=MID(A1, FIND("@",A1)+1, LEN(A1))
✅ Kab Use KaroProduct codes se category nikalna, phone numbers standardize karna, email domain extract karna.
CONCATENATE / TEXTJOIN / &
Text jodna
EXCEL
-- & se jodna (sabse simple)
=A1 & " " & B1

-- Full name banana
=A1 & " " & B1

-- Address banana
=A1 & ", " & B1 & ", " & C1

-- TEXTJOIN — separator ke saath (blank ignore karta hai)
=TEXTJOIN(", ", TRUE, A1:D1)
✅ Kab Use KaroFirst + Last name merge karna, address banana, unique key banana (City+Product).
FIND / SEARCH / SUBSTITUTE / REPLACE
Text mein dhundhna aur badalna
EXCEL
-- "@" kahan hai (case-sensitive)
=FIND("@", A1)

-- Case-insensitive dhundhna
=SEARCH("delhi", A1)

-- "Pvt Ltd" ko "Private Limited" se replace
=SUBSTITUTE(A1, "Pvt Ltd", "Private Limited")

-- Spaces hatana (sab)
=SUBSTITUTE(A1, " ", "")
✅ Kab Use KaroData standardize karna — "Pvt." ko "Private" banana, special characters hatana, text mein kuch dhundhna.

Logical Functions

Conditions ke hisaab se decision lena
IF / IFS / NESTED IF
Sabse zaroori function — condition check karna
EXCEL
-- Basic IF
=IF(B1>50000, "High", "Low")

-- Nested IF (3 conditions)
=IF(B1>=100000, "A", IF(B1>=50000, "B", "C"))

-- IFS — zyada conditions ke liye better
=IFS(
  B1>=100000, "🟢 Excellent",
  B1>=50000,  "🟡 Good",
  B1>=20000,  "🟠 Average",
  TRUE,       "🔴 Low"
)

-- IF with ISBLANK
=IF(ISBLANK(A1), "Missing", A1)
✅ Kab Use KaroPerformance category, pass/fail, high/low label, missing data flag. IFS use karo jab 3+ conditions ho — nested IF unreadable ho jaata hai.
AND / OR / NOT / IFERROR
Multiple conditions aur error handling
EXCEL
-- Dono conditions true honi chahiye
=IF(AND(A1="Delhi", B1>50000), "Yes", "No")

-- Koi bhi ek condition true ho
=IF(OR(A1="Delhi", A1="Mumbai"), "Metro", "Other")

-- Error aaye toh 0 dikhao (VLOOKUP error common hai)
=IFERROR(VLOOKUP(A1, Sheet2!A:B, 2, 0), "Not Found")

-- Division by zero error handle
=IFERROR(A1/B1, 0)
✅ Kab Use KaroIFERROR hamesha VLOOKUP ke saath lagao — warna #N/A error aata hai. AND/OR multiple conditions ke liye.

Lookup Functions — Sabse Important

Ek table se doosri table mein data dhundhna — roz kaam aata hai
VLOOKUP — Vertical Lookup
Column mein dhundhna — classic function, sabko aana chahiye
EXCEL
-- Syntax: =VLOOKUP(kya dhundho, kahan dhundho, konsa column, exact?)
=VLOOKUP(A2, Sheet2!$A:$C, 2, 0)

-- Example: Customer ID se naam dhundhna
=VLOOKUP(A2, Customers!$A:$D, 2, FALSE)

-- Error handle karte hue
=IFERROR(VLOOKUP(A2, Products!$A:$C, 3, 0), "Not Found")
⚠️ VLOOKUP ki Limitations Sirf left se right dhundhta hai. Pehle column mein hi dhundhta hai. Isliye XLOOKUP better hai (naya Excel mein available hai).
✅ Kab Use KaroOrder table mein customer naam aana, product ID se price nikalna, employee ID se department dhundhna.
XLOOKUP ⭐ — Modern Lookup (Excel 2021+)
VLOOKUP ka upgraded version — zyada powerful aur flexible
EXCEL
-- Syntax: =XLOOKUP(kya, kahan, return kya, not found)
=XLOOKUP(A2, Sheet2!A:A, Sheet2!B:B, "Not Found")

-- Left mein bhi dhundh sakta hai (VLOOKUP nahi kar sakta)
=XLOOKUP(A2, Sheet2!C:C, Sheet2!A:A, "Not Found")

-- Multiple columns return karna
=XLOOKUP(A2, Products!A:A, Products!B:D, "Not Found")
✅ Kab Use KaroNaya Excel hai toh XLOOKUP use karo — VLOOKUP se better hai har taraf. Error automatically handle karta hai.
INDEX + MATCH — Power Combo
VLOOKUP se zyada flexible — purane Excel mein XLOOKUP ka kaam karta hai
EXCEL
-- MATCH: row number dhundhta hai
=MATCH(A2, Sheet2!A:A, 0)

-- INDEX: us row ki value return karta hai
=INDEX(Sheet2!B:B, MATCH(A2, Sheet2!A:A, 0))

-- Left mein bhi kaam karta hai
=INDEX(Sheet2!A:A, MATCH(B2, Sheet2!C:C, 0))
✅ Kab Use KaroPurana Excel (2016 ya pehle) ho, ya kisi bhi direction mein lookup karna ho. Interviews mein poochhte hain — zaroori hai.

Date Functions

Date ke saath kaam — analysis mein bahut zaroori
TODAY / NOW / DATE / YEAR / MONTH / DAY
Date se parts nikalna aur banana
EXCEL
-- Aaj ki date
=TODAY()

-- Abhi ka time bhi
=NOW()

-- Date se year nikalna
=YEAR(A1)

-- Month number
=MONTH(A1)

-- Day number
=DAY(A1)

-- Kitne din purana order hai
=TODAY() - A1

-- Month name banana
=TEXT(A1, "MMMM")

-- Week number
=WEEKNUM(A1)
✅ Kab Use KaroAge calculate karna, monthly grouping, order kitne din purana hai — date-based analysis mein sab jagah.
DATEDIF / EDATE / EOMONTH / NETWORKDAYS
Date calculations — age, deadlines, business days
EXCEL
-- Age calculate karo (years mein)
=DATEDIF(A1, TODAY(), "Y")

-- 3 mahine baad ki date
=EDATE(A1, 3)

-- Mahine ka aakhri din
=EOMONTH(A1, 0)

-- Business days mein difference (weekends chhod ke)
=NETWORKDAYS(A1, B1)
✅ Kab Use KaroEmployee age/tenure, subscription expiry, delivery deadline, working days mein turnaround time.

• Too many colors
• Bina title ke
